The web app kickstarter

TL;DR

Building web apps sucks and we needed something better. We wanted an app that was easy to read, fun to write, and provided the best performance.

Here's the deal:

  • PHP driven (as in the easiest way possible)
  • Templated with Twig
  • Route-controller based, so building views is way too simple
  • Has All the whiz-bang js + css libs out there like jive, bootstrap, and modernizr

Install that puppy

Git It Bro! and then:

  • Update $config['app_root'] values to match your server's absolute path for the application
  • Make sure that templates/cache/ is writeable by apache (or the like)
  • Rage n3wbs

Requirements: PHP 5.2.4 + (Twig)

How Gus came to be

Gus was born out of frustration with the way current web apps are being built. Some are incredibly complex, robust, do-it-all systems that have a steep learning curve. Some are so bleeding edge and cool that if you get a weird bug, it's recommended to start looking into the legacy google group forums dude.

Gus gets back to the roots by using a traditional PHP server architecture and an incredibly easy template system called Twig. Additionally, Gus comes pre-packed with some industry standard client libraries like Twitter Bootstap and Modernizr, making client side development a breeze.
