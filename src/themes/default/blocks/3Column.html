Develop Easily

A modern web toolkit aimed at making your life easy. Create an app in minutes using Twig and Flat's lean routing component. Utilize frameworks like jQuery and Bootstrap, using Composer to manage all your dependencies.

Build a Flat App

A Flat App is a collection of web dependecies including HTML, JS, CSS, and Images. With just one simple command, you can generate a build and customize it to your liking.

Deploy Your App

Once your app is built, deploy it to a local environment or to the cloud using Amazon's S3 service. Flat apps run any where and require no backend services.
